# Hyprland Configuration

> A clean, modular Hyprland configuration with colors generated by matugen and Quickshell interface

## Structure

- `hyprland.conf`: Main configuration that sources all modules
- `colors.conf`: Color scheme (auto-generated by matugen)
- `config/`: Modular configuration files
  - `appearance/`: Visual settings (general, decoration, animations)
  - `system/`: Environment, autostart, and misc settings
  - `input/`: Keyboard settings
  - `dwindle.conf`: Tiling layout configuration
  - `monitors.conf`: Monitor setup
  - `workspaces.conf`: Workspace rules
  - `windowrules.conf`: Window-specific rules
  - `keybinds.conf`: All keyboard shortcuts
- `scripts/`: Utility scripts (volume control, wallpaper picker)
- `hypridle.conf`: Idle daemon configuration
- `hyprlock.conf`: Lock screen configuration

## Components

- **Display**: swww (wallpaper daemon)
- **Shell**: Quickshell (UI components and widgets)
- **Terminal**: Ghostty
- **Launcher**: bemenu
- **Clipboard**: cliphist + wl-paste

## Usage

- Reload config: `hyprctl reload`
- Pick wallpaper: `./scripts/pickwall.sh` or `SUPER + SHIFT + T`
- Lock screen: `SUPER + L` (via Quickshell)
- Toggle sidebar: `SUPER + N`
- Open cheatsheet: `SUPER + /`

| Action | Keybind |
|--------|---------|
| **System Control** | |

| Exit Hyprland | `SUPER + SHIFT + Q` |
| **Applications** | |
| Terminal (Ghostty) | `SUPER + Return` |
| Browser (Firefox) | `SUPER + W` |
| App launcher (bemenu) | `SUPER + Space` |
| Close window | `SUPER + Q` |
| Clipboard history | `SUPER + V` |
| **Quickshell Interface** | |
| Toggle sidebar | `SUPER + N` |
| Toggle cheatsheet | `SUPER + /` |
| **Window Management** | |
| Move focus | `SUPER + Arrow keys` |
| Move window | `SUPER + mouse drag` |
| Resize window | `SUPER + right mouse drag` |
| Toggle floating | `SUPER + SHIFT + F` |
| **Workspaces** | |
| Switch to workspace | `SUPER + (1-0)` |
| Move window to workspace | `SUPER + SHIFT + (1-0)` |
| Navigate workspaces | `SUPER + CTRL + Left/Right` |
| Toggle special workspace (term) | `SUPER + T` |
| **Media & Display** | |
| Volume up | `XF86AudioRaiseVolume` |
| Volume down | `XF86AudioLowerVolume` |
| Mute | `XF86AudioMute` or `SUPER + SHIFT + M` |
| Brightness up | `XF86MonBrightnessUp` |
| Brightness down | `XF86MonBrightnessDown` |
| Media controls | `XF86AudioPlay/Prev/Next` |
| **Other Tools** | |
| Change wallpaper | `SUPER + SHIFT + T` |
| Color picker | `SUPER + SHIFT + C` |
| Window screenshot | `SUPER + Print` |
| Monitor screenshot | `SUPER + SHIFT + Print` |
| Region screenshot | `Print` |

## Notes

- Do not edit `colors.conf` directly as it's auto-generated by matugen
- Quickshell handles UI elements like sidebar and lock screen
- Special workspace "term" can be toggled with `SUPER + T`
