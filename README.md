# Hyprland Configuration

> A clean, modular Hyprland configuration with colors generated by matugen

## Structure

- `hyprland.conf`: Main configuration that sources all modules
- `colors.conf`: Color scheme (auto-generated by matugen)
- `config/`: Modular configuration files
  - `appearance/`: Visual settings
  - `system/`: Environment and autostart
  - `input/`: Keyboard and gesture settings
  - `plugins/`: Plugin configurations
- `scripts/`: Utility scripts

## Usage

- Reload config: `hyprctl reload`
- Pick wallpaper: `./scripts/pickwall.sh`
- Set a lock screen: `hyprlock`
- Adjust volume: Volume keys or scripts

## Keybinds

| Action | Keybind |
|--------|---------|
| **System Control** | |
| Lock screen | `SUPER + L` |
| Exit Hyprland | `SUPER + SHIFT + Q` |
| **Applications** | |
| Terminal | `SUPER + Return` |
| Browser | `SUPER + W` |
| App launcher | `SUPER + Space` |
| Close window | `SUPER + Q` |
| Toggle side panel | `CTRL + L` |
| Clipboard history | `SUPER + V` |
| **Window Management** | |
| Move focus | `SUPER + Arrow keys` |
| Move window | `SUPER + mouse drag` |
| Resize window | `SUPER + right mouse drag` |
| Toggle floating | `SUPER + SHIFT + F` |
| **Workspaces** | |
| Switch to workspace | `SUPER + (1-0)` |
| Move window to workspace | `SUPER + SHIFT + (1-0)` |
| **Media & Display** | |
| Volume up | `XF86AudioRaiseVolume` |
| Volume down | `XF86AudioLowerVolume` |
| Mute | `XF86AudioMute` or `SUPER + SHIFT + M` |
| Brightness up | `XF86MonBrightnessUp` |
| Brightness down | `XF86MonBrightnessDown` |
| Media controls | `XF86AudioPlay/Prev/Next` |
| **Other Tools** | |
| Change wallpaper | `SUPER + SHIFT + T` |
| Color picker | `SUPER + SHIFT + C` |
| Window screenshot | `SUPER + Print` |
| Monitor screenshot | `SUPER + SHIFT + Print` |
| Region screenshot | `Print` |

## Notes

- Do not edit `colors.conf` directly as it's auto-generated
